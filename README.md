# 📊 Statistics for Machine Learning

This repository contains fundamental statistical concepts used in Machine Learning.  
Each topic is implemented in Python using practical examples and datasets.

---

## 📚 Topics

- Hypothesis Testing
- Chi-Square Test
- ANOVA (Analysis of Variance)

More statistical concepts will be added over time.

---

## 🛠️ Tools Used

- Python
- NumPy
- Pandas
- SciPy
- Matplotlib
- Seaborn

---

## 🎯 Goal

To build a strong statistical foundation required for understanding Machine Learning algorithms.
